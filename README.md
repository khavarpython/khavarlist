# KhavarList 

A personal anime and manga tracking web application built with ASP.NET Core and Azure.

##  Features

- **User Authentication** - Secure registration and login system using ASP.NET Identity
- **Anime Tracking** - Browse, search, and track your anime watching progress
- **Manga Tracking** - Keep tabs on your manga reading journey
- **Personal Stats** - View detailed statistics about your anime/manga consumption
- **Status Management** - Organize your lists by: Watching/Reading, Completed, On Hold, Plan to Watch/Read, Dropped
- **Responsive Design** - Works seamlessly on desktop and mobile devices

##  Tech Stack

- **Frontend:** ASP.NET Core MVC, Bootstrap 5, Razor Pages
- **Backend:** C# .NET 10
- **Database:** Azure SQL Database
- **Authentication:** ASP.NET Core Identity
- **Hosting:** Azure Container Apps
- **External API:** MyAnimeList (Jikan API)
